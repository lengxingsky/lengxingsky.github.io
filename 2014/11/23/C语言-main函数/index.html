C语言main()函数详解

我们在学习C语言时最先接触的就是main()函数，我们知道一个独立的C程序有且必须有一个main()函数。它是整个程序的入口，它可以调用其它的函数，但是其它的函数不能调用它。

main()函数的定义

在C99标准中main()函数只有两种定义方式

无参形式

有参形式

main()函数的位置

main()函数是可以放在程序的任何位置的，但是如果它位于被调函数之前时，必须在它前面申明被调函数，即它也遵循C语言先申明后使用的规则，但是main()函数放在被调函数之后就可以不用申明，所以我们看到的多数C程序的main()函数都放在最后。

main()函数的类型

int 是main()函数的的返回类型，类型是整型
void表示没有参数信息

main()函数的返回值

main()函数是一个程序的开始，也应该是一个程序的结束

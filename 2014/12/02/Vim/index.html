Vim的使用和配置

无论我们在Linux下修改配置文件还是编辑程序，我们都需要一款好用的编辑器，而Vim可以非常出色的完成这个任务，Vim是从vi发展出来的。据说Vim被誉为编辑器之神，足以说明它的功能强大，事实也的确如此。Vim可以用简单的操作完成复杂的任务，从而提高工作效率。在使用Vim的初期确实是困难的，但是，你越是用它就越会为它的神奇着迷。Vim的更多详情看这里。

Vim的基本操作

光标的移动（一般模式下）

用h,j,k,l,移动光标而不是上下左右的箭头

h 光标向左移动一个字符
j 光标向下移动一行
k 光标向上移动一行
l 光标向右移动一个字符

Vim中在操作命令前加数字n，代表这个操作执行的n次
如：3h 表示向左移动3个字符；23j 表示向下移动23行。

w 移动到下一个字的字首
W 移动到下一个字的字首，忽略标点符号
e 移动到下一个字的字尾
E 移动到下一个字的字尾，忽略标点符号
b 移动到上一个字的字首
B 移动到上一个字的字首，忽略标点符号
0 移动到行首(是数字0)
^ 移动到当前行的第一个非空白字符处
$ 移动到行尾(在行很长时非常好用)
f 如fa，查找当前行并移动到第一次a出现的地方
n< space > n是数字，space是空格键，表示从光标所在处向后移动n个字符
gg 移动到文件的开头，即第一行的第一个字符处
G 前面不加数字n表示：移动到文件的结尾，即最后一行的第一个字符处。加数字n表示移动到第n行
n< Enter > n是数字，表示从光标所在处向后移动n行
( 移动到当前句子的开头
) 移动到当前句子的末尾
{ 移动到当前段落的开头
} 移动到当前段落的末尾
H 移动到屏幕第一行的第一个非空白字符处
M 移动到屏幕中间的第一个非空白字符处
L 移动到屏幕最后一行的第一个非空白字符处
% 匹配括号{},[],()，如：当光标在(上时按%光标就会移到相对应的)上
Ctrl-e 向下翻一行
Ctrl-y 向上翻一行
Ctrl-d 向下翻半页
Ctrl-u 向上翻半页
Ctrl-f 向下翻一页
Ctrl-b 向上翻一页
`. 移动到最近修改过的编辑点
‘. 移动到最近修改过的位置

文本编辑操作（一般模式下）

复制、删除、粘贴、撤消、查找

y0 复制从光标到行首的内容
y$ 复制从光标到行尾的内容
yy 复制光标所在的行，yy前加数字n，表示复制从当前行开始的n行
y1G 复制当前行到第一行
yG 复制当前行到最后一行
yw 复制从光标开始的一个单词
d0 0是数字，删除从光标到行首的内容
d$ 删除从光标到行尾的内容
dd 删除当前行，dd前加数字n，表示删除从当前行开始的n行
d1G 删除当前行到第一行
dG 删除当前行到最后一行
x 删除当前的一个字符，x前加数字n，表示删除从当前字符开始的n个字符
p 将复制或删除的行粘贴在光标所在的行的下一行或将复制或删除的单词粘贴在当前光标的后面
P 将复制或删除的行粘贴在光标所在的行的上一行或将复制或删除的单词粘贴在当前光标的前面
u 撤消刚才的修改
Ctrl-r 撤消刚才的撤消
/string 从光标处向下查找名为string的字符串
?string 从光标处向上查找名为string的字符串
n 表示下一个
N 表示上一个

查找和替换(命令模式下)

在一般模式下按:即可进入命令模式

:n1,n2s/str1/str2/g 把n1行到n2行的str1替换为str2
:%s/str1/str2/g 把所有的str1替换为str2
:%s/str1/str2/gc 把所有的str1替换为str2，在替换时显示提示

进入插入模式

a 在当前字符的右边插入
A 在当前行的行尾插入
i 在当前字符处插入
I 在当前行的行首插入
o 在当前行的下一行插入
O 在当前行的上一行插入

Esc 可以退出插入模式

重复操作

Vim的宏录制

宏录制就是你把要重复的动作录制的一个寄存器里，然后Vim就照着录制的动作编辑文件。宏录制的步骤：

  1. 在一般模式下输入qa，（表示把要录制的动作存在a这个寄存器里，也可以别的名字，如b，c等),Vim的右下角会出现recording的字样
  2. 键入你想要重复的动作
  3. 键入q结束录制
  4. 在一般模式下键入@a执行刚刚录制的动作，@a前加数字n，表示执行n次

Vim块操作

v 选定要操作的内容
V 一次选定一行
Ctrl-v 可以选定任意的块

如果想在文件的每一行首加一个字符#，把光标停在首行首列，依次按：
Ctrl-v G I # Esc
如果想在文件的每一行末加一个字符$，把光标停在首行首列，依次按：
Crtl-v G $ A $ Esc
自动缩进：选定后按 =

Vim 文件操作

vim filename 打开文件，可以打开多个文件

:w 保存文件
:q 退出
:q! 不保存本次修改强制退出
:w filename 另存为
:wq 保存并退出
:r filename 把filename的内容读到光标的后面
: !command 执行Linux命令
:r !command 把command的执行结果读到光标后面

小结

Vim功能强大，基础操作也多点，但是这些操作也是有规律的，一旦掌握它们，就会觉得它们很简单。要熟悉这些基础操作，只有在平时多练习才能掌握。
